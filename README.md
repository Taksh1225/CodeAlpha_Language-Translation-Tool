This is a complete Language Translation Tool using HTML, CSS, and JavaScript with the LibreTranslate API. It includes:

 Text input
 Source & target language selection
 Translation API integration
 Display translated text
 Copy button (optional feature)

How it works:
-User enters text.
-Selects source language and target language.
-Clicks Translate.
-JavaScript sends a POST request to the translation API.
-API returns translated text.
-Result is displayed in the output box.

User can:
-Copy the translation.
-Listen to the translation using browser Text-to-Speech.
